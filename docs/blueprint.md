# **App Name**: AuthZen

## Core Features:

- Email/Password Registration: Allow users to create accounts using their email address and a password.
- Secure Login: Enable users to securely log in to their accounts.
- Password Reset: Implement a password reset feature to allow users to recover their accounts if they forget their password.
- Google Sign-In: Enable users to sign in using their Google accounts.
- Personal User Profile Page: Allow users to view and manage their profile information.
- Protected Dashboard: Create a protected dashboard accessible only to authenticated users.

## Style Guidelines:

- Primary color: Dark indigo (#4F46E5) for trust and security.
- Background color: Light gray (#F9FAFB) for a clean and modern look.
- Accent color: Purple (#A855F7) for highlights and CTAs.
- Body and headline font: 'Inter' sans-serif for a modern and readable interface.
- Use simple, consistent icons from a library like Feather Icons or Heroicons.
- Utilize a clean and responsive layout using Tailwind CSS grid and flexbox utilities. Implement DaisyUI components for a consistent and accessible design.
- Subtle transitions and animations to provide feedback to user interactions.
